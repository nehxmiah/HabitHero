01

Task Management

Core
✏️

Task Creation & Editing

Tasks are created inline with a single click or tap on any list. The input field accepts natural-language date parsing (e.g. "submit report Friday 3pm"), converting plain text into structured due dates and times automatically. Rich editing opens a detail panel with title, notes, checklists, attachments, and priority fields.

Natural Language Inline Edit Detail Panel
🔢

Priority Levels

Four priority flags — None, Low, Medium, High — displayed as colour-coded dots (grey, blue, orange, red) beside each task. Tapping the flag cycles through priorities. Tasks can be sorted or filtered by priority across any view, and high-priority items receive a subtle visual weight increase in the list.

4 Levels Color Coded Sortable
📅

Due Dates & Times

Each task supports a specific date, time, or date-range. A compact date picker overlays the list without leaving context. Overdue tasks surface in red; tasks due today are highlighted. The "Tomorrow", "Next Week", and custom date shortcuts let users reschedule in two taps.

Date Picker Overdue Alerts Quick Reschedule
🔁

Recurring Tasks

Tasks repeat on flexible schedules: daily, weekdays, weekly, monthly, yearly, or fully custom (e.g. every 3rd Tuesday). After completion, the next instance generates automatically. The recurrence badge appears inline on task rows, and editing one instance or all future instances is a clear modal choice.

Custom Cadence Edit One / All Auto-regenerate
☑️

Sub-tasks (Checklists)

Inside any task, checklist items function as sub-tasks with independent completion states. A progress ring indicator on the parent task row shows % complete at a glance. Sub-tasks can have their own due dates on Pro. Dragging reorders them; long-pressing promotes a sub-task to a standalone task.

Progress Ring Reorderable Promote to Task
🏷️

Tags

Freeform tags are added by typing # inside any task note or title. Tags live in the left sidebar as a collapsible section and act as a cross-list filter. Colour-coding tags is available on Pro, turning them into a powerful second taxonomy independent of lists.

# Syntax Cross-list Filter Color-coded (Pro)
📎

Attachments & Notes

The task detail pane has a rich-text notes area supporting inline images, URLs, and basic markdown formatting. Files (images, PDFs, audio) can be attached up to the plan's storage limit. Notes render with clean typography in a distraction-free editor experience.

Rich Text File Attach Inline Images
UX Design Note The inline editing model keeps users in flow — priority, date, and checklist fields can all be accessed without opening a full modal, reducing interaction cost significantly. The natural-language date parser is a standout onboarding accelerator.
02

Lists & Organisation

Structure
📁

Lists & Folders

Tasks live in named lists, which can be colour-coded and icon-decorated. Lists nest inside folders for a two-level hierarchy in the left sidebar. Drag-and-drop reordering is supported. Each list has an optional default start/due time, reminder, and view mode saved independently.

2-Level Hierarchy Color + Icon Per-list Settings
🔍

Smart Lists & Filters

Built-in smart lists — Today, Tomorrow, Next 7 Days, All Tasks, Assigned to Me — aggregate tasks dynamically across all lists. Users can also build custom smart lists (Pro) with multi-condition filters (priority + tag + list + date range), saved to the sidebar for instant access.

Dynamic Aggregation Custom Filters (Pro) Multi-condition
🗂️

View Modes

Each list can be viewed as a clean vertical List, a columnar Kanban board (with drag-and-drop between status columns), or a Timeline (Gantt-style bar chart for Pro). The view preference is persisted per list. Kanban columns map to task completion stages, fully renameable and reorderable.

List View Kanban Board Timeline (Pro)
↕️

Sorting & Grouping

Lists can be sorted by due date, priority, title, creation date, or manually via drag. Grouping organises tasks into visual sections by date, priority, or tag within the same list view. Groups collapse and expand, helping manage long lists without visual clutter.

5 Sort Options Group by Field Collapsible Groups
UX Design Note The Today smart list is the primary entry point for most users and is intentionally surfaced at the top of the sidebar. The Kanban view delivers a significant step-up in utility for project-minded users without complicating the default experience.
03

Calendar & Scheduling

Time
📆

Built-in Calendar View

A full-page calendar displays tasks and events in Day, 3-Day, Week, and Month views. Tasks with due times appear as time-blocks. Tasks without times appear as all-day banners at the top. Drag-and-drop on the calendar reschedules tasks instantly, with time-slot snapping for precision.

Day/Week/Month Drag Reschedule All-day Banners
🔗

Calendar Subscriptions

External calendars (Google Calendar, Outlook, Apple Calendar) are subscribed via CalDAV or iCal URL. Their events appear as read-only overlays in TickTick's calendar, giving a unified time picture alongside tasks without duplicating data.

Google Cal Outlook CalDAV / iCal
⏰

Reminders

Each task supports multiple reminders, each independently set to a specific time, a relative offset ("30 min before"), or location-based triggers (on mobile). Reminders fire as push notifications with inline snooze and reschedule actions, keeping the user out of the app for quick decisions.

Multiple per Task Location-based Inline Snooze
04

Habit Tracker

Wellbeing
🌱

Habit Creation & Frequency

Habits are separate from tasks — created in a dedicated Habits section with their own icon, colour, and goal type (check-in, count, or duration). Frequency is set per weekday combination, daily, or custom interval. Each habit has an optional target count (e.g., "drink 8 glasses").

Check / Count / Duration Custom Frequency Goal Target
📊

Streak & Statistics

A streak counter shows the current consecutive completion run and the longest streak achieved. Monthly heat maps (GitHub-style) visualise consistency over time. Completion rate percentage, total completions, and skipped days are surfaced in a per-habit statistics panel.

Streak Counter Heat Map Completion Rate
✅

Daily Check-in

The habit list is a swipeable column of today's due habits. Each row has a large tap target for completion. Count-type habits show a stepper inline. A progress bar across the top of the Habits view shows how many of today's habits are done, providing a motivating completion loop.

Swipeable Count Stepper Daily Progress Bar
UX Design Note Keeping habits architecturally separate from tasks was a deliberate design decision — habits require a different mental model (consistency over completion) and their own feedback loop. The heat-map visualisation is a high-engagement retention mechanic.
05

Focus / Pomodoro Timer

Productivity
🍅

Pomodoro Timer

The Focus section provides a built-in Pomodoro timer with configurable work intervals (default 25 min), short breaks (5 min), and long breaks (15 min). The timer displays as a large circular countdown. Each completed Pomodoro is logged against the associated task, building a time-spent history.

Configurable Intervals Task-linked Focus Log
🔔

White Noise & Focus Sounds

An ambient sound library (rain, café, forest, white noise, binaural beats) plays during focus sessions. Volume is independently controlled. On mobile, sounds continue in the background with a persistent notification showing the timer, allowing users to work in other apps while staying in a focus session.

Ambient Library Background Play Independent Volume
📈

Focus Statistics

A Focus Summary dashboard shows total focus time by day, week, and month, visualised as bar charts. Per-task focus time accumulates and is visible in the task detail. Insights surface the user's most productive time of day based on when completed Pomodoros cluster.

Time Bar Charts Per-task Log Peak Hour Insight
06

Collaboration

Team
👥

Shared Lists

Lists are shared by inviting collaborators via email or link. Permissions are set per collaborator (Can Edit vs. Read Only). Shared lists are visually distinguished in the sidebar with a collaborator avatar stack. Members see real-time updates — completed tasks disappear or are struck through simultaneously across devices.

Email / Link Invite Role Permissions Real-time Sync
👤

Task Assignment

Within a shared list, any task can be assigned to one or more members. The assignee's avatar appears on the task row. The "Assigned to Me" smart list surfaces all tasks across all shared lists where the user is the assignee, providing a unified personal responsibility view.

Multi-assignee Avatar on Row "Assigned to Me" View
💬

Task Comments

Each task has a threaded comment section accessible from the detail panel. Comments support text and image attachments. Collaborators receive push notifications on new comments. Comment timestamps and author avatars make conversation threads auditable without a separate communication tool.

Threaded Image Attach Push Notifications
🔔

Activity Feed

A notification bell surfaces a chronological activity feed of all shared list events: task created, completed, reassigned, or commented on. Each entry is tappable, deep-linking to the relevant task. The feed is per-workspace and can be filtered by list or member.

Chronological Deep Link Filterable
07

Quick Capture

Input
⚡

Global Quick Add

A system-wide keyboard shortcut (Ctrl/Cmd + Space on desktop) or a persistent floating action button (mobile) opens a minimal quick-add overlay anywhere in the app. It accepts the task name, date, and list — all in one field via natural language — and dismisses instantly, returning focus to the previous context.

Global Shortcut Float Button Zero Context Switch
🔤

Voice Input

Microphone input on mobile and desktop (where supported) lets users dictate tasks. The voice capture interface shows a pulsing waveform during recording and transcribes instantly. Siri and Google Assistant shortcuts can also create tasks in TickTick via voice from the lock screen.

In-app Dictation Siri Shortcut Google Assistant
📤

Share Extension

On iOS and Android, TickTick registers as a share target. Sharing a URL, screenshot, or text from any app pre-fills a new task with the content as an attachment or note. The extension shows a compact task creation form without leaving the source app.

iOS Share Sheet Android Intent Pre-fills Content
📧

Email to Task

Each TickTick account has a unique email address. Forwarding any email to this address creates a task with the email subject as the title and the email body as the note. On Pro, the destination list is controllable via subject-line syntax (e.g. adding #ListName).

Unique Inbox Email Subject → Title List Routing (Pro)
08

Integrations

Connected
🔌

Third-party Integrations

Native integrations exist with Slack (create tasks from messages via /ticktick command), Gmail (add task button in sidebar), Outlook add-in, Chrome extension, and Amazon Alexa. Zapier and Make (Integromat) enable no-code workflows connecting TickTick to hundreds of other services.

Slack Gmail / Outlook Zapier / Make Alexa
⌚

Wearable & Widget Support

Apple Watch and Wear OS apps provide list browsing, task completion, and quick-add from the wrist with glanceable complication widgets. iOS and Android home screen widgets display Today tasks, next due items, or a habit check-in ring, in small, medium, and large sizes.

Apple Watch Wear OS iOS / Android Widgets
🔁

Import & Export

Tasks import from Todoist, Any.do, Wunderlist, and OmniFocus via CSV or native export file. TickTick data exports as CSV or iCal format for calendar subscriptions. The iCal export updates dynamically, allowing external calendars to subscribe to a live feed of task due dates.

CSV Import/Export Todoist Migration Live iCal Feed

09

Customisation

Personalise
🎨

Themes & Appearance

Light, Dark, and Auto (follows OS) themes are available at the free tier. Pro unlocks a library of custom colour themes and accent palettes. App icons on iOS and Android are swappable. Font size is adjustable system-wide for accessibility. The sidebar width on desktop is resizable by drag.

Light / Dark / Auto Custom Themes (Pro) Font Size Custom Icon
⌨️

Keyboard Shortcuts

Desktop apps provide comprehensive keyboard navigation. A shortcut reference sheet is accessible from the Help menu. Common shortcuts: N (new task), D (set due date), P (set priority), T (today), Cmd+K (quick search), Cmd+Enter (complete task). Shortcuts are not currently remappable.

Full Navigation Shortcut Sheet Desktop Only
🔔

Notification Controls

Granular notification settings per list and system-wide: task reminders, due-date summaries (daily digest), collaboration activity, habit nudges, and Pomodoro break alerts. A "Do Not Disturb" schedule mutes all notifications between user-defined hours, surfacing a missed-reminders badge instead.

Per-list Control Daily Digest Do Not Disturb
🔒

Privacy & Security

Individual lists can be locked behind biometric authentication (Face ID, fingerprint, or PIN), hiding their content from the sidebar and requiring authentication to open. Two-factor authentication (2FA) is available for the account. All data is encrypted in transit (TLS) and at rest on Ticktick's servers.

List Locking Biometrics 2FA Encrypted
UX Design Note List locking is an underrated personal-use feature — it allows TickTick to be used for sensitive tracking (health, finance, personal goals) on a shared device without exposing private data, meaningfully broadening the app's contextual utility.
